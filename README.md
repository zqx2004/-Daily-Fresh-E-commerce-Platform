# Daily Fresh E-commerce Platform
天天生鲜电商平台（Django 实战项目）

## 项目介绍
基于 Django 4.2.4 开发的生鲜电商网站，包含用户登录注册、商品展示、购物车、订单结算、后台管理等完整功能。
适合毕业设计、求职项目展示、学习 Django 全流程开发。

## 技术栈
- 后端：Python + Django 4.2.4
- 数据库：MySQL
- 前端：HTML / CSS / JavaScript / Bootstrap
- 版本控制：Git + GitHub

## 功能模块
- 用户注册、登录、退出
- 商品分类与列表展示
- 商品详情
- 购物车添加、修改、删除
- 订单创建与支付流程
- 后台管理（商品、订单、用户）

## 运行方式
1. 克隆项目
```bash
git clone https://github.com/zqx2004/-Daily-Fresh-E-commerce-Platform.git
```
2. 安装依赖
```bash
pip install -r requirements.txt
```
3. 配置 MySQL 数据库
4. 迁移数据库
```bash
python manage.py makemigrations
python manage.py migrate
```
5. 运行项目
```bash
python manage.py runserver
```

## 项目结构
```
ttsx/
├── apps/        # 功能模块（用户、商品、订单、购物车）
├── static/      # 静态资源
├── templates/   # 页面模板
├── utils/       # 工具类
└── manage.py
```
